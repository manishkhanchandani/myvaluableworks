Indentation: always 2 spaces
Arrays:  if one element, all on the same line.  If more than one use multiple links like example below:
$templates = array(
  'content'             => 'm/404error.tpl.html',
  'my_trulia_link' => 'm/404_my_trulia_link.tpl.html'  (note last element does not have comma at end)
);
Note –2+ lines always start with 2 spaces and the => line up left padded with spaces.

Curly brackets: always new lines, not necessary for one line blocks.
eg:
If(something)
{
  line1
  line2
  etc 
}
eg2: 
If(something)
  line1


Quotes: Always use single quote where ever possible (runs a bit faster)
eg: $tpl->assign('PAGE_NAME', 'Page Not Found');

Includes:  All @ the top, and whenever possible use require_once


Please add to this list as you see fit.
